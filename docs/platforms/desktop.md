# Desktop

Desktop is the recommended starting point for most Routa users.

## Why Desktop

The desktop app gives you the full product surface in one place:

- workspace creation and switching
- provider management
- Session, Kanban, and Team flows
- local-first persistence and execution

## Install

Download the latest Desktop build for your platform from
[GitHub Releases](https://github.com/phodal/routa/releases).

## After Install

Use this order:

1. Create a workspace.
2. Open provider settings and make one provider available.
3. Attach a repository.
4. Start with a `Session` and ask Routa to inspect or plan work in that repository.
5. Move to `Kanban` when you want decomposition and lane automation.

## Best Fit

Choose Desktop when you want:

- the complete product model in one UI
- the easiest first-run path for non-contributors
- local-first storage and bundled desktop distribution
